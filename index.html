This tool builds a dev console for your GameMaker game — the little command line you open while testing to flip cheats, set values, spawn items and so on. You write your cheats in plain lines on the left, the tool turns them into one ready GML script. You paste that script once and call three functions. Done.

The idea

You type cheats like god mode = obj_player.hp = obj_player.hp_max. The box you put it in decides what kind of command it becomes — you don't click anything. Left of the = (or :) is the command you'll type in game, right of it is what it does. Empty boxes are skipped.

The five command types
  • Toggle — an on/off switch. god mode = obj_player.hp = obj_player.hp_max makes a god mode command that flips a flag, and the effect runs every frame while it's on. Leave the right side empty (show collisions =) to get just a flag you check yourself (handy for dev-only draws).
  • Set — type a value in game. set cash = global.g_cash means you type set cash 500 and it writes 500.
  • Adjust — bump a value up or down. ammo 10 = obj_player.ammo 0 99 makes add ammo and minus ammo (steps of 10, clamped 0–99).
  • Action — run code once. refill = obj_player.hp = obj_player.hp_max; obj_player.ammo = obj_player.ammo_max. Separate statements with ;.
  • Give — hand over an item by ID or name. item = obj_player.item = _id makes a give item command, so give item 5 or give item sword both work. _id is the resolved item ID. This is just an example line — change the right side to however your game adds items (an inventory array, a script, a struct, whatever). Set your items array and count under Settings.
Putting it in your game
  1. Make a new Script asset (name it scr_console) and paste the generated script into it.
  2. Pick any object that's always in the room — a dedicated obj_console or your game controller, your call. The code goes wherever you want it.
  3. On that object add three events and one call each: Create → console_create();, Step → console_step();, Draw GUI → console_draw();
  4. Run the game, press your open key (F2), type a command, hit Enter.

The console only works while DEV_MODE is true. Flip it to false for release builds so players can't cheat. obj_player in the examples is just a placeholder — use your real object and variable names. My own hand-slot setup isn't the rule, it's just one way; yours will look different and that's fine.

Write your cheats

One per line, format left = right (or use : instead of =). The box decides the type — no clicking. Leave a box empty if you don't need it.
Togglecommand = effect — flag flips on/off, effect runs every frame while ON
generates · the command flips a flag…
case "god mode":
    global.g_cheat_god_mode = !global.g_cheat_god_mode;
    _end_text = $"God mode: {draw_true_false(global.g_cheat_god_mode)}";
break;
…and the effect runs every step while it's on
if (global.g_cheat_god_mode) {
    obj_player.hp = obj_player.hp_max;
}
Setcommand = variable — type set cash 500 in game, writes the number
generates · type "set cash 500" in game
if (string_starts_with(_command, "set cash ")) {
    var _val = real(string_delete(_command, 1, 9));
    global.g_cash = _val;
    _end_text = $"Cash set to {_val}";
    return _end_text;
}
Adjustnoun amount = variable [min max] — makes add / minus
generates · "add ammo" and "minus ammo"
case "add ammo":
    obj_player.ammo += 10;
    obj_player.ammo = clamp(obj_player.ammo, 0, 99);
    _end_text = $"Ammo +10 → {obj_player.ammo}";
break;
case "minus ammo":
    obj_player.ammo -= 10;
    obj_player.ammo = clamp(obj_player.ammo, 0, 99);
break;
Actioncommand = code — runs once, separate statements with ;
generates · runs once when typed
case "refill":
    obj_player.hp = obj_player.hp_max;
    obj_player.ammo = obj_player.ammo_max;
break;
Givenoun = how to give it — type give item 5 or give item sword in game (ID or name)
generates · "give item 5" or "give item sword"
if (string_starts_with(_command, "give item ")) {
    var _arg = string_delete(_command, 1, 10);
    var _id  = -1;
    if (string_digits(_arg) == _arg) {
        _id = real(_arg);                        // number = ID
    } else {
        for (var _i = 0; _i < ITEM_ID.END; _i++) {  // text = find by name
            if (string_lower(global.g_items[_i].name) == _arg) { _id = _i; break; }
        }
    }
    if (_id >= 0 && _id < ITEM_ID.END) {
        obj_player.item = _id;
        _end_text = $"Gave {global.g_items[_id].name}";
    }
    return _end_text;
}
